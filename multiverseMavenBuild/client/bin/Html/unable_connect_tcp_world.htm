MULTIVERSE WORLD BROWSER ERROR

Unable to Connect to Server

The Multiverse World Browser was unable to connect to the specified world server. Possible causes include:

  • The world server is not running - If this is your server, check server status and log files.
  • Incorrect world entry in Multiverse master server database - If the world is under your control, check the entry in the Multiverse master server database.
